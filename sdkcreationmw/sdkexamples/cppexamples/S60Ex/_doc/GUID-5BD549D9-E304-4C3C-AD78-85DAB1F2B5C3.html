About the example applications

Selected examples

The examples have been chosen to demonstrate a range of Symbian OS and S60 object-oriented design and programming techniques.

Documentation

Documentation is provided for all example applications. The documentation consists of an overview of the application as well as class and file documentation.

Application-specific documentation is located in <S60_SDK_installation_directory>\S60CppExamples\<example_application>\doc\index.html.

Symbian OS and S60 versions

The examples are relevant to Symbian OS version 9.4 and the S60 user interface package.

Coding conventions

As far as possible, the example code follows relevant C++, Symbian OS and S60 coding conventions. A full list of the conventions is given in the Coding conventions.

Building and running the examples

The examples are all built in the same way. Instructions on building and running them are provided in Building example applications for an S60 device.
