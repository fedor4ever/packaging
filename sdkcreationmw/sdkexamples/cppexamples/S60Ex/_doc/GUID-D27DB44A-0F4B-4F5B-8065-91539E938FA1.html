Prerequisites

Before you start building and running example applications, you should have the following installed on your workstation:

  • Active Perl version 5.6.1

    You can download ActivePerl from the ActivePerl website. See ActivePerl documentation for installation instructions.

  • Java Runtime version 1.5.0 or newer

    Java Runtime can be downloaded from Sun Developer Network Downloads. (JRE is included in J2SE.)

  • Nokia Symbian^3 SDK

  • S60 example applications - delivered with the SDK installation package and installed to <S60_SDK_installation_directory>\S60CppExamples by default)

For further details on prerequisites, please refer to the Nokia Symbian^3 SDK Installation Guide.
