List of example applications

  • AddressBook - provides an example, including source code, of the use of the Nokia Web Services Framework so that developers can test and run the application against a live server.

  • Animation - demonstrates how to use animation in client/server architecture.

  • Application Interworking Example - demonstrates how to use the AIW Consumer APIs.

  • AudioStreaming Example - demonstrates how to use Audio Streaming API.

  • Browser Control - demonstrates how to create a browser control that will enable you to display web related content. The example application also demonstrates how to customize the browser control to fit the unique needs of your host application, by showing you how to customize soft keys, dialogs, scroll bars, etc.

  • Chat - demonstrates how to create a application that sends messages from one Bluetooth or TCP/IP device to another.

  • Client/Server Example (asynchronous) - demonstrates the architecture of a simple client server application utilising asynchronous calls to the server

  • Client/Server Example (synchronous) - demonstrates the architecture of a simple client server application utilising synchronous calls to the server

  • CommonDialogsEx - demonstrates the usage of Common Dialogs APIs.

  • Contacts Database - demonstrates how to use Contacts database.

  • Data Mobility Example - demonstrates the use of Connection Settings UI and Mobility extension of Connection Manager API

  • Directory Localizer Example - demonstrates the usage of DirectoryLocalizer API.

  • DriveInfo Console Application - demonstrates how to create a console application with icon.

  • Dynamic SettingList Example - demonstrates how the create setting item lists dynamically.

  • Filelist - demonstrates how basic grid can be implemented in the Symbian OS and use it to filelisting with graphics.

  • FindItem TestApp - demonstrates Find Item API functionality. The Find Item API offers methods for parsing phone numbers, e-mail addresses or URI addresses from given text.

  • Focus Event - demonstrates how to utilize focus-events.

  • Graphics - demonstrates two methods of drawing bitmaps to the screen.

  • Gui/Engine Split - demonstrates how to split an application into two components, an engine dll and a GUI application.

  • HapticsExample

  • Hello World Basic - demonstrates how to create a simple interactive application in the Symbian OS.

  • HierarchicalColumnList - demonstrate the usage of hierarchical column list UI component. The application creates a hierarchical list, adds some data items to the list, and then allows the user to browse the list structure and change some of the list setting through options menu.

  • HWRMTestApp - demonstrates HW Resource Manager API functionality. HW Resource Manager provides client APIs for controlling lights and vibra of the device. It also has additional responsibillty of publishing certain power state related Publish & Subscribe properties.

  • IsvTelCallApp - demonstrates how to use the ISV ETel APIs.

  • IsvTelInfoApp - demonstrates how to use the ISV ETel APIs.

  • Listbox - demonstrates how to use Listbox control and middle softkey (MSK).

  • Localization - demonstrates how to localize an application to adapt to different languages.

  • Landmarks Reference Application - demonstrates Landmark reference application development.

  • Landmarks UI Reference - demonstrates the usage of the selector and editor dialogs exported by the Landmarks UI applicatio.

  • Location Reference - demonstrates how to use the Mobile Location Framework to obtain position information.

  • Messaging - demonstrates the use of SendUI API for creating SMS, MMS and email messages.

  • MyView - demonstrates the use of different user interface views.

  • Netscape Plugin Control - demonstrates the use of Browser Plug-in API.

  • Note - demonstrates the use of different kind of notes.

  • Optical Character Recognition - demonstrates the use of Optical Character Recognition API.

  • Touch Examples:

    • ChoiceList - demonstrates the use of CAknChoiceList. Using the Options menu, user can create choice lists and edit created choice list content.

    • LongTapDetector - demonstrates the use of CAknLongTapDetector. Receives pointer events from owner application or owner control and returns location if a long tap event occurs.

    • NoughtsAndCrosses - demonstrate scalability and the usage of bitmaps and svg-t graphics .

    • StylusPopupMenu - demonstrates the use of CAknStylusPopUpMenu.A floating popup menu which is meant for stylus usage only.

    • TactileExample - demonstrates how to use Tactile Feedback Client API on Symbian OS application.

  • SensorExample - SensorExample is an example application that demonstrates Sensor API functionality. Sensor API provides clients the access to the data provided by various Sensors of the device.

  • SIP - explains the Session Initiation Protocol (SIP) Example Application. The SIP Example Application is a simple game for two players, but it is a complete application, which demonstrates the use of the SIP API to set up a connection to a remote device.

  • Unit Editor - demonstrates how to use a multi field numeric unit editor

  • Webclient - demonstrates how to create simple HTTP Client software.

  • XeDomExample - demonstrates, how to use XML Engine Document Object Model APIs for creating, parsing, processing and destroying of XML documents and its variety of different components with which the document is formed.
