Building example applications for an S60 device

In order to install an application to an S60 device, you need to do the following:

  1. Compile the application for the target by creating a GCCE UREL build of it.

  2. Create a Symbian installation (.sis) file of the GCCE build.

  3. Sign the .sis file.

  4. Install the .sis file to your S60 device
